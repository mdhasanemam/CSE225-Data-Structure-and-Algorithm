# CSE225 - Data Structure and Algorithm

This repository contains all course materials, theory practice codes, and lab works for the **CSE225: Data Structure and Algorithm** course. It is mainly created for learning, practicing, and revising different DSA concepts and problem-solving techniques.

---

## Repository Structure

### 📁 Theory Class Materials
This folder contains:
- Class PPT slides
- Lecture PDFs
- Additional theory resources

These files help in understanding the theoretical concepts taught during the course.

---

### 📁 Theory Code Practice
This folder includes:
- Theory class practice codes
- Personal DSA practice problems
- Implementations of different algorithms and data structures
- Basic to intermediate problem-solving codes

---

### 📁 Lab Work
This folder contains:
- All lab tasks and solutions
- Mid exam practice files
- Final exam practice files
- Additional lab resources

> ⚠️ The `Lab Work` folder size is around **10.2 MB**.  
> To access all files properly, it is recommended to download the repository.
